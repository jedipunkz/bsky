# CLAUDE.md

## Pull Request Guidelines

- Write all PRs in **English**
- After making changes, always: **commit → push → create PR using `gh` command**

### PR Body Structure

Use the following sections in every PR:

```
## Context

[Background and motivation for this change]

## Summary

[Brief description of what this PR does]

## What I did

- [Bullet points describing specific changes made]
- [...]

## Reference

- [Links to related issues, docs, or external resources]
```
